Overview

After activating the theme and previewing the homepage, you may wonder that your website is already filled in with built-in content and questioning about how to customize it.

WPortfolio provides huge possibilities to customize the theme to meet our needs. For an easier and best practice to customize the theme, it's highly recommended to do by using a child theme.

There are two ways to customize the theme, by overriding the templates nor using WordPress hooks.

Overriding Templates

We can override the parent template by creating a PHP file with the same name as the parent's inside our templates directory in our child theme.

For example, in our parent theme, we have wp-content/themes/wportfolio/templates/front-page/masthead.php, we can override it by creating an exact file in our child theme, so we'll have wp-content/themes/wportfolio-child/templates/front-page/masthead.php

This can be achieved because the parent theme will look up the possible template in child theme first before looking in parent theme.

Hooks

It is highly recommended to use hooks to customize the theme, since it has less effort than overriding templates. In order to support that, WPortfolio provides a lot of hooks either action or filter to customize the theme to meet our needs.
